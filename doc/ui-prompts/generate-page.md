# 生成页面 UI 提示词

## 页面概述
生成页面是用户配置手写参数并生成手写图片/PDF的核心交互页面。

## 表单设计

### 文本输入区域
- **位置**: 页面顶部左侧
- **组件**: 多行文本框 + 文件上传
- **尺寸**: 宽度 `400px`，高度 `200px`
- **功能**: 
  - 直接输入文本
  - 上传文档文件（.doc, .docx, .pdf, .txt, .rtf）
  - 实时字符计数
  - 文本预处理（去除特殊字符）

### 参数配置面板
- **布局**: 多列布局，自动换行
- **分组**:
  - 基础参数（字体、字号、行间距）
  - 页面参数（宽度、高度、边距）
  - 扰动参数（可展开）
  - 段落排版（对齐、缩进、段间距）

### 字体选择组件
- **下拉框**: 显示可用字体列表
- **上传按钮**: 支持自定义字体上传
- **预览**: 鼠标悬停显示字体预览
- **状态**: 选中状态高亮显示

### 纸张模板选择
- **模板类型**: 空白、单线、方格、点阵
- **颜色选择**: 预设颜色 + 自定义颜色
- **纸张规格**: A4、A5、B5、Letter
- **预览**: 实时预览纸张效果

## 交互流程

### 生成流程
1. 用户输入/上传文本
2. 配置生成参数
3. 点击"预览"或"生成"按钮
4. 显示加载状态
5. 等待任务完成（WebSocket/轮询）
6. 显示结果/下载文件

### 状态管理
- **生成中**: 禁用所有按钮，显示进度
- **冷却中**: 显示倒计时，禁用按钮
- **队列满**: 显示等待时间，禁用按钮
- **完成**: 启用按钮，显示结果

## 实时预览

### 预览区域
- **位置**: 页面右侧
- **尺寸**: 最大宽度 `600px`
- **功能**:
  - 单页预览
  - 多页导航（上一页/下一页）
  - 图片缩放
  - 下载按钮

### 预览触发
- **自动预览**: 参数变化时自动刷新（可选）
- **手动预览**: 点击"预览"按钮
- **全量预览**: 开发环境专用

## 状态提示

### 进度显示
- **上传进度**: "内容正在上传并处理…"
- **生成进度**: "任务处理中（X%）"
- **排队状态**: "前方排队 X 人，当前排队 X 人"
- **WebSocket**: 实时状态更新

### 错误处理
- **网络错误**: Toast 提示，自动重试
- **服务器错误**: Toast 提示，显示错误信息
- **参数错误**: 表单验证，高亮错误字段
- **队列满**: 显示等待时间，自动重试

## 下载功能

### 图片下载
- **格式**: PNG
- **文件名**: `handwriting.png`
- **多页**: ZIP 压缩包
- **触发**: 点击下载按钮

### PDF 下载
- **格式**: PDF
- **文件名**: `handwriting.pdf`
- **多页**: 自动分页
- **触发**: 点击"生成 PDF"按钮

## 响应式设计

### 移动端（< 768px）
- **布局**: 单栏布局
- **表单**: 全宽显示
- **预览**: 全宽显示
- **按钮**: 垂直排列，全宽

### 平板端（768px - 1024px）
- **布局**: 两栏布局
- **表单**: 左侧，宽度 `50%`
- **预览**: 右侧，宽度 `50%`

### 桌面端（> 1024px）
- **布局**: 两栏布局
- **表单**: 左侧，最大宽度 `650px`
- **预览**: 右侧，自适应宽度

## 可访问性

### 键盘导航
- **Tab 顺序**: 从上到下，从左到右
- **快捷键**: Enter 提交表单，Esc 取消操作
- **焦点状态**: 清晰的焦点轮廓

### 屏幕阅读器
- **标签**: 所有表单元素有关联标签
- **提示**: 错误信息有 ARIA 提示
- **状态**: 动态内容有 ARIA live 区域
