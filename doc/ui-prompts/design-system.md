# 手写文字生成网站 - 整体设计规范

## 颜色系统

### 主色调
- **主色**: `#007BFF` (Bootstrap Primary Blue)
- **成功色**: `#28A745` (Bootstrap Success Green)
- **警告色**: `#FFC107` (Bootstrap Warning Yellow)
- **危险色**: `#DC3545` (Bootstrap Danger Red)
- **信息色**: `#17A2B8` (Bootstrap Info Cyan)

### 中性色
- **深色文字**: `#2C3E50` (主要文字)
- **中等文字**: `#6C757D` (次要文字)
- **浅色文字**: `#ADB5BD` (辅助文字)
- **背景色**: `#F8F9FA` (页面背景)
- **白色**: `#FFFFFF` (卡片背景)
- **边框色**: `#DEE2E6` (分隔线、边框)

### 状态色
- **禁用状态**: `#CCCCCC` (按钮禁用)
- **悬停状态**: `#0056B3` (按钮悬停)
- **激活状态**: `#003D73` (按钮按下)

## 字体系统

### 字体家族
- **主字体**: `Avenir, Helvetica, Arial, sans-serif`
- **手写字体**: `KaiTi, STKaiti, FangSong, serif` (用于预览效果)
- **代码字体**: `Monaco, Menlo, Consolas, monospace`

### 字号规范
- **标题 H1**: `2rem` (32px)
- **标题 H2**: `1.5rem` (24px)
- **标题 H3**: `1.25rem` (20px)
- **正文**: `1rem` (16px)
- **小字**: `0.875rem` (14px)
- **辅助文字**: `0.75rem` (12px)

### 行高
- **紧凑行高**: `1.2`
- **标准行高**: `1.5`
- **宽松行高**: `1.8`

## 间距系统

### 基础间距单位
- **4px**: 最小间距（图标与文字间距）
- **8px**: 小间距（组件内部间距）
- **16px**: 中等间距（组件之间间距）
- **24px**: 大间距（区块之间间距）
- **32px**: 超大间距（页面边距）

### 组件间距
- **按钮间距**: `10px` (margin-right)
- **表单间距**: `10px` (margin-bottom)
- **卡片内边距**: `20px` (padding)
- **页面边距**: `20px` (padding)

## 圆角系统

### 圆角大小
- **小圆角**: `4px` (输入框、小按钮)
- **中圆角**: `5px` (普通按钮、卡片)
- **大圆角**: `8px` (弹窗、大卡片)
- **圆形**: `50%` (头像、圆形按钮)

## 阴影系统

### 阴影层次
- **轻微阴影**: `0 2px 5px rgba(0, 0, 0, 0.1)` (卡片、输入框)
- **中等阴影**: `0 4px 20px rgba(0, 0, 0, 0.15)` (弹窗、下拉菜单)
- **强烈阴影**: `2px 2px 4px rgba(0, 0, 0, 0.5)` (按钮按下状态)

## 动画系统

### 过渡时间
- **快速过渡**: `0.2s` (按钮悬停)
- **标准过渡**: `0.3s` (大部分交互)
- **慢速过渡**: `0.5s` (页面切换)

### 缓动函数
- **标准缓动**: `ease`
- **加速缓动**: `ease-in`
- **减速缓动**: `ease-out`
- **加减速缓动**: `ease-in-out`

## 响应式断点

### 断点定义
- **手机**: `< 576px`
- **平板**: `576px - 768px`
- **桌面**: `768px - 992px`
- **大桌面**: `992px - 1200px`
- **超大桌面**: `> 1200px`

## 组件规范

### 按钮
- **主要按钮**: 蓝色背景，白色文字，圆角 `5px`
- **次要按钮**: 灰色背景，深色文字
- **危险按钮**: 红色背景，白色文字
- **禁用状态**: 灰色背景，灰色文字，禁止点击

### 输入框
- **边框**: `1px solid #ddd`
- **圆角**: `5px`
- **内边距**: `10px`
- **聚焦状态**: 蓝色边框，轻微阴影

### 卡片
- **背景**: 白色
- **边框**: 无
- **阴影**: 轻微阴影
- **圆角**: `5px`
- **内边距**: `20px`

## 可访问性

### 对比度
- **文字与背景**: 至少 `4.5:1` 对比度
- **大文字与背景**: 至少 `3:1` 对比度

### 焦点状态
- **键盘导航**: 清晰的焦点轮廓
- **屏幕阅读器**: 适当的 ARIA 标签
