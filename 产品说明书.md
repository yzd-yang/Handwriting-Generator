# 手写文字生成网站 - 产品说明书

## 一、产品简介

手写文字生成网站是一款在线工具，能够将用户输入的文字转换成为逼真的手写体图片或 PDF 文件。通过模拟真实手写的随机扰动效果（字号抖动、位置偏移、角度旋转、删除线、墨水深浅等），生成的文字看起来如同真人手写一般自然，适合作业、论文、信件、笔记等多种场景。

- **版本**：前端 v1.4.0 / 后端 v1.0.0
- **技术架构**：Vue 3 前端 + FastAPI 后端，支持 Docker 一键部署
- **访问方式**：部署后通过浏览器访问

## 二、核心功能

### 2.1 富文本编辑器

内置 Tiptap 富文本编辑器，支持以下格式操作：

| 功能 | 说明 |
|------|------|
| 粗体 | 选中文字加粗 |
| 斜体 | 选中文字倾斜 |
| 下划线 | 选中文字添加下划线 |
| 无序列表 | 创建项目符号列表 |
| 有序列表 | 创建编号列表 |
| 段落对齐 | 左对齐 / 居中 / 右对齐 / 两端对齐 |

编辑器工具栏固定在顶部，内容区域可滚动，支持长文本编辑。

### 2.2 文档导入

支持导入以下格式的文档，自动保留段落格式：

- `.docx` — Word 文档（保留标题、段落、缩进等格式）
- `.pdf` — PDF 文档
- `.txt` — 纯文本文件
- `.rtf` — 富文本格式

### 2.3 多种笔记模板

网站提供四种纸张模板，满足不同书写需求：

| 模板类型 | 说明 | 适用场景 |
|---------|------|-----------|
| 空白 | 纯色底，无任何线条 | 自由书写、绘画 |
| 单线 | 经典横线纸，线条间距可调 | 作业、信件、日常笔记 |
| 方格 | 横竖等距方格，格距 = 行间距 | 中文写作、数学作业、绘图 |
| 点阵 | 等距圆点网格，点距 = 行间距 | bullet journal、排版设计 |

### 2.4 多种纸张颜色

内置 6 种预设颜色，支持自定义任意颜色：

- 纯白（默认）
- 护眼黄
- 淡蓝
- 浅灰
- 纯黑
- 淡紫

点击色块选中预设颜色，或点击颜色选择器自定义任意颜色。选中状态有明显的高亮边框和放大效果。

### 2.5 多种纸张规格

一键切换常用纸张尺寸（宽度 × 高度，单位：像素 @300dpi）：

| 规格 | 尺寸（px） | 说明 |
|------|------------|------|
| A4 | 2481 × 3507 | 默认，国际标准 |
| A5 | 1754 × 2480 | 便携笔记本 |
| B5 | 2953 × 4169 | 日系笔记本常用 |
| Letter | 2550 × 3300 | 美式信纸 |

也可以手动输入任意宽度和高度，自由定制纸张大小。

### 2.6 丰富的字体支持

- 内置多种手写字体可供选择
- 支持上传自定义字体文件（`.ttf` / `.otf`）
- 可调节字号大小

### 2.7 自然手写效果

通过多种随机扰动参数，让生成的文字呈现真实手写的不规则感：

- **字号抖动**：每个字的字号有微小随机变化
- **位置偏移**：每个字在 x / y 方向有微小位移
- **角度扰动**：每个字有微小旋转角度
- **删除线**：可设置随机删除线（概率、长度、角度、宽度、墨水深浅均可调）
- **中英文间距**：中英文混排时自动调整间距

### 2.8 段落排版

- **对齐方式**：通过富文本编辑器工具栏设置（左对齐 / 居中 / 右对齐 / 两端对齐）
- **首行缩进**：可设置缩进字符数（0 = 不缩进）
- **段间距**：可设置段间额外空行数

### 2.9 导出功能

- **预览**：点击预览按钮，快速渲染第一页查看效果
- **生成图片**：生成全部页面，打包为 ZIP 文件下载
- **导出 PDF**：生成全部页面，合并为 PDF 文件下载

## 三、使用指南

### 步骤 1：输入文字

在首页富文本编辑器中输入或粘贴需要生成的文字内容。支持：

- 直接输入文字
- 使用富文本工具栏设置格式（粗体、斜体、对齐等）
- 点击"选择文件"导入 Word、PDF、TXT 等文档

### 步骤 2：选择字体

- 从下拉菜单中选择喜欢的字体
- 或点击"选择文件"上传自定义字体（`.ttf` 或 `.otf` 格式）

### 步骤 3：设置纸张

1. **选择模板**：点击"空白 / 单线 / 方格 / 点阵"中的一个
2. **选择颜色**：点击色块选中预设颜色，或点击右侧颜色选择器自定义颜色
3. **选择规格**：点击"A4 / A5 / B5 / Letter"快速设置纸张大小，也可手动输入宽度和高度

### 步骤 4：调整参数

根据喜好调整以下参数（均有默认值，可直接使用）：

- **字号**：手写字体大小
- **行间距**：各行之间的距离
- **四边距**：上、下、左、右页边距
- **首行缩进**：段首缩进字符数
- **段间距**：段间额外空行数
- **随机扰动参数**：调节手写自然度（sigma 值越大，变化越明显）

### 步骤 5：生成并下载

- **预览**：点击"预览"按钮，快速查看第一页效果
- **生成图片**：点击"生成图片"按钮，生成全部页面并下载 ZIP
- **导出 PDF**：点击"生成 PDF"按钮，生成全部页面并下载 PDF

## 四、参数详细说明

### 基础参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 字体 | 选择内置字体或上传自定义字体 | — |
| 字号 | 字体大小（px） | 124 |
| 行间距 | 行与行之间的距离（px） | 200 |
| 上边距 | 页面顶部留白（px） | 50 |
| 下边距 | 页面底部留白（px） | 50 |
| 左边距 | 页面左侧留白（px） | 50 |
| 右边距 | 页面右侧留白（px） | 50 |

### 段落排版参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 首行缩进 | 段首缩进字符数 | 2 |
| 段间距 | 段间额外空行数 | 0 |

> **提示**：对齐方式通过富文本编辑器工具栏设置，支持左对齐、居中、右对齐、两端对齐。

### 随机扰动参数（高级）

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 字号抖动 Sigma | 字号随机变化幅度 | 2 |
| 行距抖动 Sigma | 行间距随机变化幅度 | 0 |
| 字间距 | 每个字之间的额外间距（px） | 1 |
| 字间距 Sigma | 字间距随机变化幅度 | 2 |
| X 偏移 Sigma | 每个字水平方向随机偏移幅度 | 3 |
| Y 偏移 Sigma | 每个字垂直方向随机偏移幅度 | 3 |
| 角度扰动 Sigma | 每个字旋转角度随机变化幅度 | 0.05 |

> **建议**：将各 Sigma 参数设置为 `1~3`，可以获得较为自然的手写效果；设置过大可能导致文字难以辨认。

### 删除线参数（高级）

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 删除线概率 | 每个字出现删除线的概率（0~1） | 0.005 |
| 删除线长度 Sigma | 删除线长度随机变化幅度 | 2 |
| 删除线角度 Sigma | 删除线倾斜角度随机变化幅度 | 2 |
| 删除线宽度 Sigma | 删除线粗细随机变化幅度 | 2 |
| 删除线宽度 | 删除线基础宽度（px） | 8 |
| 墨水深浅 Sigma | 删除线颜色深浅随机变化幅度 | 30 |

## 五、常见问题

### Q1：生成的文字看起来不够"手写"？

尝试增大随机扰动参数（字号抖动、位置偏移、角度扰动的 Sigma 值），推荐各 Sigma 设置为 `1~3`。

### Q2：深色纸张上看不清文字？

在高级参数中将"墨汁深度标准差"调大，或选择浅色纸张。

### Q3：可以生成多少页？

单次生成页数取决于文本长度和行间距设置。使用预览模式可先查看第一页效果，确认后再导出完整文件。

### Q4：支持哪些字体格式？

支持 `.ttf` 和 `.otf` 格式的字体文件。

### Q5：生成的 PDF 可以用于打印吗？

可以。生成图片和 PDF 的分辨率适用于屏幕显示和日常打印。如需高质量打印，可适当增大字号和纸张尺寸。

### Q6：导入 Word 文档后格式会保留吗？

会。导入 `.docx` 文件时，系统会自动保留段落结构、标题格式等信息，并在富文本编辑器中显示。

### Q7：预览和生成图片有什么区别？

预览只渲染第一页，速度更快，适合快速查看效果。生成图片会渲染全部页面并打包为 ZIP 文件。

## 六、技术说明

### 系统要求

- 现代浏览器（Chrome / Firefox / Edge / Safari）
- 后端需 Python 3.10+ 运行环境
- 建议使用 Docker 部署，一键启动前后端

### 部署方式

详见项目根目录下的 `docker-compose.yml`，执行以下命令即可启动：

```bash
docker compose up -d
```

启动后：
- 前端访问：`http://localhost:2345`
- 后端 API：`http://localhost:5005`

### 核心技术

- **手写渲染**：基于 `handright` 库，支持丰富的随机扰动效果
- **图像处理**：Pillow（PIL）用于背景图生成和图像合成
- **文档导出**：PyPDF2 用于 PDF 合成
- **异步任务**：支持多任务排队，避免并发过载
- **富文本编辑**：基于 Tiptap 编辑器，支持格式化文本输入
- **文档解析**：pypandoc 用于 Word 文档解析，保留段落格式
