# 主页 UI 提示词

## 页面概述
主页是用户进入网站后的第一个页面，包含手写文字生成的核心功能表单和预览区域。

## 布局结构

### 整体布局
- **布局方式**: CSS Grid 两栏布局
- **左侧**: 表单区域（参数配置）
- **右侧**: 预览区域（实时预览）
- **响应式**: 移动端单栏布局

### 表单区域
- **宽度**: 最大 `650px`
- **内边距**: `20px`
- **阴影**: 轻微阴影
- **多列布局**: 自动多列（列宽 `200px`）

### 预览区域
- **宽度**: 自适应
- **内边距**: `20px`
- **阴影**: 轻微阴影
- **图片**: 最大宽度 `600px`，居中显示

## 表单组件

### 文本输入区
- **标签**: "文字:"
- **输入框**: 多行文本框，高度 `200px`
- **占位符**: "请输入要转换的文字"
- **文件上传**: 支持 `.doc, .docx, .pdf, .txt, .rtf`

### 字体选择
- **标签**: "字体文件:"
- **按钮**: "选择文件"（蓝色，`#4285f4`）
- **下拉框**: 字体列表，宽度 `60%`
- **样式**: 蓝色背景，白色文字

### 背景图片
- **标签**: "背景图片文件:"
- **按钮**: "选择文件"（禁用状态：灰色）
- **清除按钮**: 右下角 X 按钮
- **加载状态**: 旋转加载动画

### 尺寸设置
- **宽度/高度**: 数字输入框
- **禁用状态**: 有背景图片时禁用
- **清除按钮**: 右侧 × 按钮

### 纸张模板
- **模板类型**: 空白、单线、方格、点阵
- **按钮样式**: 圆角按钮，选中状态高亮
- **颜色选择**: 预设颜色 + 自定义颜色选择器
- **纸张规格**: A4、A5、B5、Letter

### 参数设置
- **字体大小**: 数字输入框，推荐 > 100
- **行间距**: 数字输入框
- **边距**: 上下左右四个数字输入框
- **扰动参数**: 可展开的高级设置区域

### 段落排版
- **对齐方式**: 下拉选择（默认、居中、右对齐）
- **首行缩进**: 数字输入框
- **段间距**: 数字输入框

## 操作按钮

### 按钮组
- **载入设置**: 从 localStorage 加载预设
- **保存设置**: 保存当前设置到 localStorage
- **重置设置**: 恢复默认值
- **预览**: 生成预览图
- **生成完整手写图片**: 生成完整图片
- **生成 PDF**: 生成 PDF 文件
- **反馈**: 跳转到反馈页面

### 按钮状态
- **正常状态**: 蓝色背景，白色文字
- **悬停状态**: 深蓝色，轻微放大
- **按下状态**: 更深蓝色，轻微缩小
- **禁用状态**: 灰色背景，禁止点击

## 预览区域

### 图片预览
- **默认图片**: `/default1.webp`
- **图片样式**: 最大宽度 `100%`，圆角边框
- **多页预览**: 上一页/下一页导航

### 页数提示
- **显示条件**: 生产环境且有文本内容
- **样式**: 警告提示框
- **内容**: 预计页数 + 限制说明

## 状态提示

### 生成状态
- **生成中**: 蓝色背景，动画效果
- **冷却中**: 橙色背景，倒计时显示
- **队列满**: 警告提示，倒计时

### 消息提示
- **成功消息**: 绿色 Toast
- **错误消息**: 红色 Toast
- **信息消息**: 蓝色 Toast
- **上传消息**: 蓝色 Toast，持续显示

## 交互效果

### 悬停效果
- **输入框**: 轻微放大，阴影加深
- **按钮**: 背景色变深，轻微放大
- **下拉框**: 轻微放大，阴影加深

### 加载效果
- **文件上传**: 旋转加载动画
- **图片生成**: 进度提示
- **页面跳转**: 过渡动画

## 响应式设计

### 移动端（< 1000px）
- **布局**: 单栏布局
- **表单**: 全宽显示
- **预览**: 全宽显示
- **按钮**: 垂直排列

### 桌面端（≥ 1000px）
- **布局**: 两栏布局
- **表单**: 左侧，最大宽度 650px
- **预览**: 右侧，自适应宽度
